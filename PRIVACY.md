# Política de Privacidade - PoE Item Search

## Resumo
A extensão PoE Item Search não coleta, armazena ou transmite dados pessoais de usuários.

## O que a extensão faz
- Armazena dados localmente no navegador do usuário (localStorage do Chrome)
- Acessa apenas os sites pathofexile.com/trade e pathofexile.com/trade2
- Permite que o usuário pesquise itens e mantenha histórico de buscas

## Dados coletados
**Nenhum dado pessoal é coletado ou transmitido.**

Os seguintes dados são armazenados APENAS localmente no computador do usuário:
- Histórico de buscas de itens
- Filtros salvos
- Marcadores de itens

## Como os dados são armazenados
- Todos os dados são armazenados no localStorage do navegador Chrome
- Nenhum servidor externo recebe esses dados
- Os dados são deletados quando o usuário limpa os dados de navegação do Chrome
- A extensão não tem acesso a dados de outras abas ou extensões

## Permissões utilizadas
- `storage`: Para armazenar histórico e preferências localmente
- `host_permissions` (pathofexile.com/*): Para funcionar no site de trade oficial

## Conformidade
Esta extensão está em conformidade com as políticas de privacidade do Chrome Web Store e LGPD (Lei Geral de Proteção de Dados).
